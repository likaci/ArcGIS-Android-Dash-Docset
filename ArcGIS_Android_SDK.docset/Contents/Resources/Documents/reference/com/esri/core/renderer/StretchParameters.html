Class Overview

StretchParameters are options of the StretchRenderer and the RGBRenderer.

List of StretchParameters that extend this class are:

  • StretchParameters.ClipStretchParameters
  • StretchParameters.HistogramStretchParamaeters
  • StretchParameters.MinMaxStretchParameters
  • StretchParameters.StdDevStretchParameters
Example: 
 StdDevStretchParameters params = new StretchParameters.StdDevStretchParameters();
 params.setStdDev(2.0f);
