Class Overview

Contains information on a URL used to access a resource when using REST to access an OGC service. This will consist of the type of resource (eg. "Tile" or "FeatureInfo"); the mime type of the resource; a template URL that can be used to access the resource. The template url will consist of replaceable tokens delimited with '{' and '}'.
