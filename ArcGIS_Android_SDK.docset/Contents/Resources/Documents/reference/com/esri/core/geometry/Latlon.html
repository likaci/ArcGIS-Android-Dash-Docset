public static Point[] createPoints (double[][] latlon)

Creates points from latitude and longitude coordinates.

Parameters
latlon an array of one or more latitude and longitude pairs, e.g. {{37.761, -122.50}, {37.750, -122.474}, ...}. latitude and longitude are measured in degrees.
Returns
  • an array of points or null if the given latlon is null or empty.

public static Polygon createPolygon (double[][] latlon)

Creates a polygon from latitude and longitude coordinates.

Parameters
latlon an array of one or more latitude and longitude pairs, e.g. {{37.761, -122.50}, {37.750, -122.474}, ...}. latitude and longitude are measured in degrees.
Returns
