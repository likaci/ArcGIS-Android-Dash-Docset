Class Overview

This class represents a spatial filter which includes an input geometry and spatial relationship. This serves as an input for queries with spatial constraints.

You can for example construct a spatial filter using:

  • An Envelope geometry & CONTAINS
  • A Polygon geometry & INTERSECTS
