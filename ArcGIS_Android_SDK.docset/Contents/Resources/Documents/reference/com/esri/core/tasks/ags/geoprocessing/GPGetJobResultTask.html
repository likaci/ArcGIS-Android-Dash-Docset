Class Overview

This class implements a task that can be passed to a thread pool or thread to asynchronously retrieve an output parameter from an asynchronous geoprocessing task. This will return either the result data or a result image depending on the values passed in the GPGetResultParameters object used to create this instance. In both cases, a GPParameter will be returned via the callback.
