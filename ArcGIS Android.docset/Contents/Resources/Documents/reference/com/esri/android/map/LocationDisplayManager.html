Class Overview

Class providing UI support to display the current location of the device on a MapView. To get an instance of a LocationDisplayManager see getLocationDisplayManager().

public float getNavigationPointHeightFactor ()

Gets the navigation point height factor, which controls the height of the location graphic when using ERROR(/AutoPanMode.NAVIGATION). This is because navigation typically has the point centered near the bottom, so you can see as much of the road ahead as possible. Values should be between 0.0 (the bottom of the screen) and 1.0 (the top of the screen).

Returns
  • the height at which to center the location graphic (default 0.125f)

Gets the wander extent factor, which affects autopanning. If autopanning is ERROR(/AutoPanMode.LOCATION), this property determines how far from the center you may pan the map before it will center on the location again. Only works with ERROR(/AutoPanMode.LOCATION) return the factor of how far you may pan away before autopanning back. (default 0.5f)
